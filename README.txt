Análise de Dados de Voos na Índia

Este projeto realiza a análise de dados de voos de companhias aéreas na Índia, com base em um conjunto de dados extraído de um arquivo .csv disponível no Kaggle.

Tecnologias utilizadas:
- Python, Jupyer Notebook
- Bibliotecas: Seaborn, Pandas, Matplotlib, Streamlit, NumPy

Problemas de negócio:
- Analisar as variações de preços de passagens aéreas para identificar padrões de comportamento de voos.
- Analisar relações geográficas e temporais dos voos dentro da India.

Metodologia:
- Extração e limpeza dos dados: Tratei valores faltantes, transformei alguns dados e padronizei nomes.
- Análise Exploratória: Feita inteiramento no Jupyter notebook, com o objetivo de criar insights relevantes.
- Visualização: Utilizando a biblioteca Streamlit, criei Dashboards gráficos para melhorar a visualização.

Como executar:
 Clone este repositório:
  git clone https://github.com/seu-usuario/seu-projeto.git

Instale as dependências:
- pip install pandas matplotlib streamlit seaborn numpy

Execute o projeto com Streamlit:
- streamlit run app.py

Fonte dos dados
Os dados foram obtidos a partir de um dataset público em https://www.kaggle.com/datasets/rohitgrewal/airlines-flights-data.
